# おでかけMAP 作業記録

作業日: 2026-04-08

## 実施内容

- `GitHub Pages` で公開できる静的構成の埋め込み用 Web アプリを作成
- `Google Maps JavaScript API` を使った地図表示、カテゴリ切替、一覧表示、詳細表示を実装
- API キー未設定時でも閲覧エラーにならないフォールバック表示を実装
- `?embed=1` で余白を抑えた埋め込み向け表示を追加
- 投稿フォーム URL と修正フォーム URL の設定状況を画面上に表示する仕組みを追加
- GitHub Pages 公開用に `.nojekyll`、README、公開チェックリスト、設定見本を整備
- ローカル検証用に `tools/validate-data.mjs` を作成

## 今回の追加要件への対応

- 境界ポリゴンを詳細データへ差し替え
- 境界データを単一ポリゴンだけでなく複数ポリゴン構造でも扱えるよう改修
- フォーム経由で任意追加されたカテゴリをデータから自動認識し、表示フィルタに反映できるよう改修
- タイトル「千葉ニュータウンのおすすめを見つける」のフォントを柔らかい印象の書体へ変更
- スポット詳細におすすめ写真のカルーセル表示を追加
